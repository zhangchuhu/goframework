# 使用说明

```bash
$ cd $GOPATH/src/code.yy.com/bilin
$ git clone https://code.yy.com/bilin/bilingo.git
$ mv $GOPATH/src/code.yy.com/bilin/bilingo $GOPATH/src/bilin
```

# 进程功能

* adpromotion 广告
* apigateway http的api网关
* appspider 抓取竞品数据
* bcserver 直播间
* bcserver_timer 直播间定时任务
* bigexpression 大表情
* bizroomcenter 
* ccserver 呼叫控制
* chattag 随机聊打标签, rest接口
* chattagtars 随机聊打标签, tars接口
* clientcenter 对tars和dubbo接口的封装
* common 通用函数
* confinfocenter 配置服务
* flowerserver 送花
* guildtars 工会
* livingrecoam 实时录音管理
* matchserver 随机匹配算法
* official 官频
* onlineperson 在线的人
* operationManagement 运营管理
* protocol protobuf协议定义
* rcserver 没有用
* relationlist 亲密榜
* roominfocenter 房间信息服务
* searchserver 搜索服务
* thrift 协议定义
* userinfocenter 用户信息服务

# 代码编译

进入进程目录
```bash
make tar
```
就能生成发布包。参考tars开源框架的上线方式。
